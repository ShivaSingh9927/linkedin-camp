**Strategies:**
✅ 1. FOLLOW BUTTON
🔍 Best selector strategy

Use aria-label (most stable):

const followBtn = page.locator('button[aria-label^="Follow"]').first();
✅ Why it works
aria-label="Follow Vijay Sarathy" → predictable prefix
Independent of UI redesigns
⚠️ Edge case

If already followed:

label becomes "Following ..."

👉 handle it:

if (await followBtn.isVisible()) {
    await followBtn.click();
}
✅ 2. LIKE BUTTON (on post)
🔍 Best selector
const likeBtn = page.locator('button[aria-label*="Reaction button"]').first();
💡 Better logic (avoid double-like)
const likeBtn = page.locator('button[aria-label*="Reaction button"]').first();

const label = await likeBtn.getAttribute('aria-label');

if (label.includes('no reaction')) {
    await likeBtn.click();
}
✅ Why
LinkedIn encodes state in aria-label
"no reaction" → not liked
"Like" or "Celebrate" → already reacted
✅ 3. COMMENT BUTTON
🔍 Best approach (DON’T click this anchor directly)

That <a> opens a new page, not inline comment.

✅ Better strategy
const commentBtn = page.locator('button:has-text("Comment")').first();
await commentBtn.click();
Then type:
const commentBox = page.locator('[contenteditable="true"]').last();
await commentBox.click();
await page.keyboard.type("Nice post!", { delay: 50 });
✅ 4. CONNECT BUTTON
🔍 Best selector
const connectBtn = page.locator('a[aria-label^="Invite"][href*="custom-invite"]').first();
✅ Flow after click
await connectBtn.click();

// Wait for modal
const sendBtn = page.locator('button:has-text("Send")').first();
await sendBtn.click();
⚠️ Edge cases
Sometimes button is hidden inside "More"
Sometimes already connected → button not present

👉 fallback:

const moreBtn = page.locator('button:has-text("More")').first();
✅ 5. CONTACT INFO
🔍 Selector
const contactBtn = page.locator('a[href*="overlay/contact-info"]').first();
⚠️ Important reality

Even if you click:

Email/phone may be hidden
Depends on:
connection level
privacy settings
NOT just premium

👉 handle gracefully:

if (await contactBtn.isVisible()) {
    await contactBtn.click();
}

const postText = await page.locator('[data-testid="expandable-text-box"]').first().innerText();
