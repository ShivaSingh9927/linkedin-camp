# Template Thumbnail Prompts 
## Style Reference (prepend to every prompt)

> Modern flat vector illustration, clean professional style, indigo (#4f46e5) and slate (#475569) color palette, tech/business theme, simple composition with plenty of negative space, subtle gradient background, no text

---

## My Network (9 templates)

### 1. Contextual Reactivation
> A dormant LinkedIn connection icon glowing back to life, a subtle handshake between two profiles connected by a dotted thread, indigo pulse rings radiating outward

### 2. Deep Context Multi-Touch
> A trail of 8 glowing footsteps on a curved path — each step a different icon: eye, clock, heart, chat bubble, clock again, chat bubble, clock, chat bubble — with the last step brightest

### 3. Direct Content Engager
> A heart icon (like) inside a speech bubble, floating above a LinkedIn post card, with a message arrow connecting them. Two-step minimal composition

### 4. Observer to Authority
> A figure standing behind a veil, then stepping forward into the light, with a like icon, comment icon, and message icon forming an upward staircase

### 5. Omni-Channel Sync
> A central profile icon with two outward arrows: one pointing to an email envelope, one pointing to a CRM database cylinder. Data flow lines connecting all three

### 6. Quick Email Pivot
> A LinkedIn profile with a fast-switch arrow redirecting away to an email envelope. Bypass visual — LinkedIn crossed out softly, email glowing

### 7. Silent Data Harvester
> A stealthy figure in indigo silhouette visiting a LinkedIn profile, data particles flowing invisibly to a database vault. No message icons present

### 8. Two-Step Nurture
> Two chat bubbles in sequence — first one small, second one larger — connected by a dotted wait line with a clock icon in between

### 9. Warm Engager Loop
> A circular loop of like and comment icons alternating: heart → speech → heart → speech → chat bubble. Cyclic arrow connecting the last back to the first

## Out of Network (6 templates)

### 10. Blank Slate Connector
> A completely empty LinkedIn profile with a single glowing "Connect" button. Clean, minimal, one-step composition

### 11. Direct Email Route
> A LinkedIn profile with an arrow pointing diagonally away to an email icon, no message bubbles at all. Email as the only destination

### 12. Fluff-Free Intro
> A streamlined arrow going directly from a profile visit icon to a connect icon. Zero intermediary steps. Minimal two-icon composition

### 13. Omni-Channel AIDA
> A funnel with 5 layers: eye, handshake, clock, email icon, database cylinder. Each layer a distinct color gradating from light indigo to deep slate

### 14. PAS Sync
> Three panels: Problem (broken cog), Agitation (exclamation mark), Solution (lightbulb) — with a sync arrow flowing to a CRM database. Soothing indigo-to-slate gradient

### 15. Warm-Up Connection
> A thermometer-style warmup: cold blue profile → lukewarm like icon → warm comment icon → hot red connect button. Temperature gradient rising
