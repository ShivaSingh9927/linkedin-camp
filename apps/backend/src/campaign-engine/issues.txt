1. 🔴 No Stealth Plugin (Bot Detection)
File: engine.ts line 1

The campaign engine imported plain playwright while your working testscripts used playwright-extra + stealth plugin. Without stealth, LinkedIn's bot detection flagged the browser instantly and invalidated the session → login redirect.

diff
- import { chromium } from 'playwright';
+ import { chromium } from 'playwright-extra';
+ const stealth = require('puppeteer-extra-plugin-stealth')();
+ chromium.use(stealth);
